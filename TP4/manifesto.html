Resumo:

Este projeto baseia-se na criação de um servidor em Java Script capaz de navegar por vários ficheiros através da analise do link do browser.

Um dos principais objetivos deste projeto passa pela criação de vários ficheiros jason. Cada um destes ficheiros contém toda a informação referente a uma região de Portugal. Outro dos objetivos trata-se da criação de um servidor, utilizando uma porta, de tal forma que seja possivel aceder ao "index.html" assim como todos os ficheiros html das regiões de Portugal através dessa porta. Para tal é necessário fazer um parse ao link de modo a saber qual o ficheiro html pretendido e abri-lo.
